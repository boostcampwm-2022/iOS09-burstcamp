  ### **부스트캠프가 끝나도 우리의 성장은 끝나지 않았다!**    
  
  <br/>
  
  부스트캠퍼들의 동기부여와 지속적인 발전을 위한 커뮤니티, **burstcamp** 입니다.    
  자신의 성장을 공유하고 싶지만, 부끄럽거나 바쁜 캠퍼들을 위해 블로그 글을 자동으로 공유합니다.   
  앱을 통해 캠퍼들의 다양한 소식들을 접하며 동기부여를 얻고, 내적 친밀감을 쌓을 수 있습니다.
